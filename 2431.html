Course
Description:		 Introduction to operating system concepts: process, CPU scheduling, memory management, file system and storage, and multi-threaded programming.
Course
Objectives:		 Upon satisfactory completion of the course, the student will be:
  • competent with process concepts and CPU scheduling
  • competent with memory hierarchy and memory management
  • familiar with process control blocks, system calls, context switching, interrupts, and exception control flows
  • familiar with process synchronization, inter-process communication, and threads
  • familiar with multi-threaded programming
  • familiar with file systems and disk scheduling algorithms and I/O
  • Be exposed to security
Suggested Texts: 1) Silberschatz, Glavin, Gagne, Operating Systems Concepts (9th or 10th Edition)
2) (Optional) Bryant, O'Hallaron, Computer Systems: A Programmer's Perspective
